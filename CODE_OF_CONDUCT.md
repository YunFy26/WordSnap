# Code of Conduct

WordSnap 希望为所有参与者提供友善、专业且不受骚扰的协作环境，无论其经验、身份、背景或表达方式如何。

## 我们期待的行为

- 就事论事，批评代码和方案，不攻击个人。
- 假设善意，清楚说明上下文，并给新贡献者学习和修正的空间。
- 尊重不同观点、使用场景和语言习惯。
- 接受维护者对范围、优先级、安全性和可维护性的最终判断。
- 发现错误时及时更正；受到影响时先停止有害行为。

## 不可接受的行为

- 骚扰、歧视、威胁、侮辱、跟踪或持续的人身攻击。
- 未经允许公开他人的私人信息、通信、API Key 或选中文本。
- 性暗示内容、不受欢迎的关注，或任何让他人感到不安全的行为。
- 垃圾信息、恶意破坏，以及明知违反项目安全要求的披露。

## 适用范围与执行

本规范适用于仓库、Issue、Pull Request、讨论区以及代表 WordSnap 参与的公开空间。维护者可以编辑或删除不当内容，拒绝贡献，暂时限制互动，或永久禁止严重及持续违规者参与。

如需报告，请优先使用仓库提供的私密联系方式。若暂时没有私密通道，可创建一个不包含敏感细节的 Issue，请维护者建立私人联系。维护者会尽量保护报告者和受影响者的隐私，并根据事件严重程度采取一致、适当的措施。
